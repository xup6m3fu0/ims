# 產品預設資料匯入

## 檔案放置位置

將 Excel 檔放在此資料匣（`backend/data/import/`）內，檔名請使用：

- **products.xlsx**（建議）
- 或 **products.xls**（舊版 Excel）

## 初始檔案格式（最少兩欄）

- **第一個工作表**會被讀取。
- **第一列必須是標題列**。

**必填欄位（至少要有這兩個）：**

| 中文標題（擇一） | 英文標題（擇一） |
|------------------|------------------|
| 客戶名稱         | customerName     |
| 產品名稱         | productName      |

- **編號**：不必在 Excel 裡填，匯入時會依序自動產生（P-0001、P-0002、…）。
- **其他欄位**（庫存數量、庫位、狀態、備註）：檔案裡可以沒有；沒有則不寫入、使用預設值，之後可在系統內修改。
  - 預設：庫存數量 = 0，庫位 = A，狀態 = 成品，備註 = 空。

### 範例（第一列為標題，只建兩欄即可）

| 客戶名稱 | 產品名稱     |
|----------|--------------|
| 宏達公司 | 齒輪箱（大） |
| 宏達公司 | 齒輪箱（小） |
| 星航科技 | 控制板 v2    |

若有加其他欄位，標題可為：編號/code、庫存數量/quantity、庫位/location、狀態/status、備註/note。庫位限 A～E，狀態限 成品／半成品／不良品。

## 執行匯入

在專案根目錄（與 `package.json` 同層）：

```bash
npm run import:products
```

或進入後端目錄：

```bash
cd backend
npm run import:products
```

- 會讀取 `backend/data/import/products.xlsx` 或 `products.xls`。
- 編號依現有資料序號往後自動帶入；其餘列新增至產品表。
